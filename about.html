About The Project

The AI CAREER RECOMMENDATION SYSTEM is a frontend-based academic project developed using HTML,CSS, and JAVASRIPT. The objective of this project is to demonstrate how Artificial Intelligence concepts such as rule-based systems and decision logic can be implemented on the client side.

Unlike traditional system,this project does not depend on a backend or database,making it lightweight and fast.

Technologies Used:


• HTML- Structure and layout

• CSS- Styling and responsiveness

• JavaScript- Rule-based AI logic and interacton
